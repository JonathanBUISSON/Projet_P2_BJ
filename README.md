git clone https://github.com/JonathanBUISSON/Projet_P2_BJ.git

Création de l'environnement virtuel
python -m venv c:\path\to\myenv

Sélectionner P2_Code_Source_BUISSON_Jonathan
puis lancer avec "Run"
